Anthropic's postmortem on a rough stretch for Claude Code users in early 2025. Three separate issues compounded: a reasoning effort default change in March, a caching bug that caused context loss, and a verbosity-reduction prompt that degraded coding output. All three were identified and fixed by April 20, with usage credits reset for affected subscribers.
