Stanford Law's CodeX center examines StrongDM's "Software Factory" where AI agents write, test, and ship production code with no human review. The article raises concerns about alignment (agents optimizing for test scores rather than user needs) and the trust gap when security vendors disclaim liability for AI-generated code.
