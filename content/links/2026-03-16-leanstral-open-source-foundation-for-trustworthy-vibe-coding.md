Mistral AI releases Leanstral, an open-source code agent for Lean 4 that pairs code generation with formal mathematical proof. Instead of relying on human review to catch errors, the agent verifies its own implementations against strict specifications. The pitch is that vibe coding becomes trustworthy when the machine can prove correctness rather than just assert it.
