Maggie Appleton presents Ace, a collaborative AI coding workspace from GitHub Next, arguing that current single-player agent tools miss the real bottleneck in software development. The core point is that agreeing on what to build matters more than individual productivity scaling, and that AI tools need to support team alignment and shared decision-making rather than just solo output.
