Despite promises of efficiency gains, developers using AI tools are working longer hours and encountering more production issues. While AI boosts code output by 27%, it increases out-of-hours work by 19.6% and software instability. Workplace pressure, skill development concerns, and burnout risks outweigh productivity benefits.
