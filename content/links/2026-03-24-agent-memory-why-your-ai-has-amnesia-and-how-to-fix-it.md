AI agents forget everything between sessions, starting each conversation from scratch. The author maps out four memory types borrowed from human cognition (working, procedural, semantic, episodic) and argues that implementing them at scale is a database problem. Vectors, graphs, relational data, and ACID transactions all need to work together - stitching separate databases for each type creates brittle systems with consistency and security gaps.
